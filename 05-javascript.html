ES6+ Modern JavaScript Features ES6+
What this demonstrates: ES6 (ECMAScript 2015) and later versions introduced modern syntax that makes JavaScript more readable and powerful: arrow functions, destructuring, template literals, spread operator, const/let, and array methods.
ARROW FUNCTIONS
// Old way:
function add(a, b) {
  return a + b;
}

// Arrow function:
const add = (a, b) => a + b;
TEMPLATE LITERALS
const name = "Alice";
const age = 25;

// Old way:
name + " is " + age;

// Template literal:
`${name} is ${age}`
DESTRUCTURING
const user = {
  name: "Bob",
  age: 30
};

// Destructure:
const { name, age } = user;
ARRAY METHODS
const nums = [1,2,3,4,5];

nums.filter(n => n > 2);
// → [3, 4, 5]

nums.map(n => n * 2);
// → [2, 4, 6, 8, 10]
