Section 01 — Overview

Full Stack Development

Understanding the layers of a modern web application — from what the user sees to how data is stored and retrieved.

Definition

Full stack development refers to the practice of building both the frontend (client-side) and backend (server-side) of a web application. A full stack developer has skills across every layer of the stack — from designing what a user sees in their browser, to writing server logic, to managing databases. The term "stack" refers to the collection of technologies used together to build a complete application.

The Three Layers

Every web application is built from three distinct layers. Click each layer to explore it.

Layer 1 — Client Side
Frontend
Everything the user sees and interacts with. Runs in the browser. Responsible for layout, styling, user interactions, and presenting data from the backend.
HTML CSS JavaScript React Bootstrap
⇅
Layer 2 — Server Side
Backend
The application logic that runs on a server. Handles requests from the frontend, processes business rules, talks to the database, and returns responses (usually JSON).
Node.js Express Python / Django REST API Authentication
⇅
Layer 3 — Persistence
Database
Where data is stored permanently. The backend queries the database to read, write, update, and delete data. Can be relational (SQL) or document-based (NoSQL).
MySQL PostgreSQL MongoDB SQLite Firebase

How a Request Works

A typical journey from a user clicking a button to data appearing on screen.

01

User Interaction

User clicks a button or submits a form in the browser (Frontend).

02

HTTP Request

JavaScript sends an HTTP request (GET/POST) to the backend server via an API endpoint.

03

Server Processing

The backend receives the request, validates it, and runs business logic.

04

Database Query

The server queries the database — fetching or updating the required data.

05

JSON Response

Server sends data back to the browser as a JSON response.

06

DOM Update

JavaScript uses the response to update the page without a full reload.

Real-World Tech Stacks

Different combinations of technologies are used depending on the project requirements.

MERN Stack

A JavaScript-only stack popular for SPAs and real-time applications like social platforms.

MongoDB · Express · React · Node.js

LAMP Stack

A classic stack used for CMS sites, blogs and e-commerce. Powers much of the early web.

Linux · Apache · MySQL · PHP

Django Stack

Python-based stack favoured for data-heavy applications and rapid development.

Python · Django · PostgreSQL · JS

JAMstack

Modern serverless approach using static builds, APIs and CDN delivery. Very fast.

JavaScript · APIs · Markup
