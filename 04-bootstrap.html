Section 04 — Framework

Bootstrap 5

Bootstrap is a front-end CSS/JS framework that provides a responsive grid system, pre-built components, and utility classes. All examples below use real Bootstrap 5.

The Grid System BOOTSTRAP GRID
What this demonstrates: Bootstrap's grid system divides the page into 12 columns. Use .container for a centered wrapper, .row for rows, and .col-* for columns. Responsive breakpoints (col-sm-*, col-md-*, col-lg-*) change the layout at different screen widths. Columns automatically stack on mobile.

Bootstrap Components — Cards COMPONENTS
What this demonstrates: Bootstrap cards (.card) are flexible content containers. They include .card-header, .card-body, .card-title, .card-text and .card-footer sub-components. Cards can contain any HTML and can be placed in a grid layout using Bootstrap's row/col.

Buttons, Badges & Alerts UI COMPONENTS
What this demonstrates: Bootstrap provides pre-styled UI components using contextual colour classes: btn-primary, btn-success, btn-danger etc. These classes apply consistent colour meanings across buttons, badges, alerts and more. Use btn-outline-* for bordered variants.

Bootstrap Utility Classes UTILITIES
What this demonstrates: Bootstrap's utility classes apply single CSS properties directly via class names, removing the need to write custom CSS for common patterns. Examples include spacing (m-*, p-*), display (d-flex, d-none), text alignment (text-center), and more.
