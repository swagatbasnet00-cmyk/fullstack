CSS Custom Properties (Variables) CSS VARIABLES
What this demonstrates: CSS custom properties (variables) are defined with a --name syntax inside a :root selector, making them globally accessible. Used via var(--name). They make theming, consistency and updates dramatically easier — change one value to update the whole page.
--bg
#0a0a0f — Page background colour
--accent
#7b5ea7 — Primary accent colour
--highlight
#00ffc2 — Secondary highlight
--muted
#6b6b80 — Secondary text

Responsive Design & Media Queries RESPONSIVE
What this demonstrates: Media queries use @media (max-width: ...) to apply different styles at different viewport sizes. This makes layouts adapt to mobile, tablet and desktop screens. Combined with flexible units like %, vw, rem and clamp(), pages respond fluidly.
📱 Mobile — @media (max-width: 480px) { font-size: 14px; columns: 1; }
💻 Tablet — @media (max-width: 768px) { font-size: 15px; columns: 2; }
🖥️ Desktop — Default styles — columns: 3; font-size: 16px;
